{{page.title}}

The value of a field can be calculated in real-time in the browser as the user interacts with the form.

The value is calculated and displayed in the browser for fast feedback but recalculated on the server, to prevent client-side manipulation, before insertion into the DataDocument.

A calculated field is specified using the rf.calculated attribute. This attribute value should be a function which returns the calculated value. The function has access to the fields map.

Calculated fields are evaluated in the order that they appear in the form.

<form rhinoforms="true">
  <input name="date-of-birth" type="text" />
  <!-- Calculate age -->
  <!-- Seconds in a year is 31536000000 -->
  <input name="age" type="text" disabled="disabled"
    rf.calculated="Math.floor((new Date() - new Date(fields.date-of-birth.value)) / 31536000000)" />
  <input type="submit" rf.action="next" value="Next" />
</form>
