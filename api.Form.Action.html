In Rhinoforms all Form navigation is done using Actions. The actions available to a form and their behaviour is defined in the Form Flow.

An action can be bound to a button or other element using the rf.action attribute, it can then be triggered by clicking that element.

An example using a submit button: 

<input type="submit" rf.action="next" value="Next" />

An example using an A link: 

<a href="#" rf.action="next">Next</a>

The benefits of a Submit Button

To get the best user experience in your forms it's a good idea to use a submit button rather than an A link or anything else. This is because most browsers will let you submit a HTML form using the enter key.

If a form has more than one action available make the most natural action an input of type 'submit' and make any others of type 'button'. Most browsers will trigger the first 'submit' button on the page when the user hits enter.
