In Rhinoforms all Form navigation is done using Actions. The actions available to a form is defined in the Form Flow.

An action can be bound to a button or other html element using the rf.action attribute, it can then be triggered by clicking that element.

An example using a submit button: 

<input type="submit" rf.action="next" value="Next" />

An example using an A link: 

<a href="#" rf.action="next">Next</a>

Tip: Use a Submit Button for the Next Action

To get the best user experience in your forms it's a good idea to use an input of type 'submit' for the 'next' action. This will allow the user to sumbit the form using the enter key.

Use an input of type 'button' for any other actions. Most browsers will trigger the first 'submit' button in the DOM when the user hits enter.

Action Parameters

Most actions don't need parameters, but for those which do parameters the parameter name and value should be given after a question mark: 

<a href="#" rf.action="editEntity?index=5">Edit</a>

Xpaths can be used: 

<a href="#" rf.action="editEntity?index={{//entity/index}}">Edit</a>

Multiple parameters should be seperated with an ampersand: 

<a href="#" rf.action="editEntity?type=fish&index=5">Edit</a>
