A form page can use an include to inject commonly used HTML from another file.

An include statement is declared using the rf.include tag. The tag should have a src attribute with the relative path to the include file.

<form rhinoforms="true">
  <input name="name" type="text" />
  <rf.include src="address-block.html" />
  <input type="submit" rf.action="next" value="Next" />
</form>
