A Validation Function is a short JavaScript function which provides additional logic on top of Validation Keywords. The presence and value of other fields in the current form can be used to decide what, if any, Validation Keywords should apply.

Validation Functions provide validation for a single field and are declared using rf.validationFunction.

In this example text in the 'languages' field is only required if 'multilingual' is checked. 

Can you speak more than one language?
  <input name="multilingual" type="checkbox" />

What languages can you speak?
  <input name="languages" type="text"
    rf.validationFunction="if (fields.multilingual.value == 'yes') { this.validate('required'); }" />

Fields Object

The JavaScript function can make use of the fields object. The fields object contains an entry for every field in the current form mapped by field name. Each field entry has the following properties:

  • name - (string) the name of the field with any full-stop characters replaced with an underscore.
  • value - (string) the current value of the field.
  • included - (boolean) the result of any rf.includeIf statements. Defaults to true.

As with Validation Keywords, Validation Functions are run in the browser and again on the server for bullet-proof input validation.
