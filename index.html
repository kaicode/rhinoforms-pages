Rhinoforms

Collect and manipulate complex data sets using forms with minimal code.

A cleaner, simpler alternative to XForms.

View the Docs »

Develop forms rapidly using familiar HTML.

Use <form> and <input> elements to build your forms.

A sprinkling of Rhinoforms syntax adds power.

For example, simple validation: 

<input name="email" rf.validation="required email" >

Syntax also enables:

  • Custom JavaScript Validation Functions
  • Form/Content Generation using your Data
  • Conditional Inclusion Logic
  • Includes

Ultra Secure.
Validation runs in the browser for speed and again on the server for security.

Each form can target a specific area of your XML data set.

Build complex data sets using a Form Flow.

  • Drill down through your Data
  • Capture/Maintain Sub-Entities

Use External Web Services to Load, Enrich or Persist your data.
