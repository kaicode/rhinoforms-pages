Rhinoforms

Make Fast, Secure Forms with minimal code

Download this project as a .zip file Download this project as a tar.gz file

Rhinoforms is a secure, lightweight HTML Forms Platform that enables the capture of structured data sets against strict validation using forms with minimal markup.

Rhinoforms is designed to support rapid Forms development using HTML and JavaScript without the need for server-side coding.

  • Forms Validated in the Browser for Speed
  • Forms Validated again on the Server for Security (using the same code!)
  • Custom Validation Functions
  • Non-linear Form-Flows
  • Data Captured as XML
  • Integrate with other Web Services
  • Capture Repeating Entities
  • Cleaner, Simpler Alternative to XForms
